# Stats Preview Card Component

## Descripción
Reto de Frontend Mentor: una tarjeta que combina una imagen ilustrativa con overlay de color y un bloque de estadísticas (texto + números destacados), con layouts distintos para móvil y escritorio.

## Qué practiqué
- Enfoque **mobile-first**: primero se maqueta la versión móvil y luego se adapta a desktop con media queries, en vez de al revés.
- **Flexbox** para reordenar los bloques (imagen y texto cambian de posición/dirección según el breakpoint).
- Superposición de color sobre una imagen usando `background-color` + `mix-blend-mode` (o pseudo-elemento) para lograr el overlay del diseño.
- Jerarquía tipográfica clara entre los números grandes y las etiquetas descriptivas.

## Tecnologías
HTML5, CSS3 (Flexbox, mobile-first)

## Aprendizaje clave
Reforcé el flujo de trabajo **mobile-first**, que ayuda a priorizar el contenido esencial antes de añadir complejidad para pantallas grandes.
